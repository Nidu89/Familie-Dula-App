Die Familie-Dula-App ist eine private Familienorganisations-Plattform (solo-developer, kein festes Budget/Deadline).

**Tech Stack:** Next.js App Router, Supabase (Auth/DB/Storage/Realtime), Tailwind CSS, shadcn/ui

**Rollen:** Admin, Erwachsene, Kinder (mit eingeschränkten Rechten via RLS)

**MVP (P0):** PROJ-1 Auth, PROJ-2 Familienverwaltung, PROJ-3 Dashboard, PROJ-4 Kalender, PROJ-5 Aufgaben, PROJ-6 Belohnungssystem

**P1:** PROJ-7 Einkaufslisten, PROJ-8 Essens-/Rezeptplanung, PROJ-9 Chat, PROJ-10 Benachrichtigungen

**P2:** PROJ-11 Bild-Upload im Chat, PROJ-12 iCloud Kalender Integration

**Why:** Privates Familienprojekt für die Familie Dula zur Alltagskoordination (Termine, Aufgaben, Einkäufe, Kommunikation).

**How to apply:** Immer RLS-Sicherheit berücksichtigen, Kinder-taugliche UI, Realtime-Features via Supabase bevorzugen.
